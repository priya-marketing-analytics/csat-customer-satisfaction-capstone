# E-Commerce Customer Service Satisfaction Analysis

## Overview
Analyzed 85,000+ e-commerce customer service interactions to identify
drivers of customer satisfaction (CSAT) and build predictive models.

## Business Objective
- Identify service factors influencing CSAT
- Predict high vs low satisfaction
- Segment customers for service improvement

## Tools & Techniques
- Python (Pandas, Scikit-learn)
- Logistic Regression, Gradient Boosting
- Clustering & Segmentation
- Tableau, Excel

## Key Insights
- Resolution time and agent tenure strongly influence CSAT
- Certain issue categories consistently drive low satisfaction
- Predictive models achieved strong ROC-AUC performance

## Deliverables
- Cleaned datasets
- Python notebooks
- Model evaluation outputs
- Business recommendations
